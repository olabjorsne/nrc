To link the "nrc" module directly into node-red.

1. in the directory containing the node’s package.json file, run: (sudo) npm link.
[Tomas] On Windows, I ran without sudo but had opened the cmd terminal as an administrator

2. in your node-red user directory, typically ~/.node-red run: npm link nrc
[Tomas] On Windows, I ran without sudo but had opened the cmd terminal as an administrator

(3. Installed the nrc module using "npm install nrc" but not sure it is needed)
